Step 1: Connect Your Display

Set up your e-ink display with WiFi credentials. If you don't have a display, you can skip this step and use the web dashboard only.

Requires flashing with Commute Compute custom firmware (CCFirm)

WiFi Configuration

Option A: Connect via Bluetooth (Recommended)

Send WiFi credentials directly to your display via Bluetooth. No captive portal needed.

Network list will appear after connecting to device

Requires Chrome or Edge browser. Your display must show "DEVICE SETUP" screen.

OR use pairing code
Option B: Enter Display Pairing Code

If Bluetooth isn't available, enter the 6-character code shown on your display (requires WiFi already configured on device).

Code expires after 10 minutes.

Factory Reset Device

Clear all stored WiFi credentials and pairing data from your device. Use this if you need to reconfigure or move to a new network.

No device? No problem! You can still use the web dashboard to view your commute information on any browser.
